# Loan Default Prediction - React Frontend

Modern React frontend for the Loan Default Prediction system.

## Setup

1. **Install dependencies:**
```bash
npm install
```

2. **Start development server:**
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Build for Production

```bash
npm run build
```

## Environment Variables

If you need to change the API URL, edit `src/App.jsx` and update the `API_URL` constant.

## Features

- Clean, modern UI with form validation
- Real-time predictions via FastAPI backend
- Visual risk indicators (Low/Medium/High)
- Responsive design for mobile and desktop
- All loan features input fields
